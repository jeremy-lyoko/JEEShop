# 一.项目简介
本项目主要使用了JavaWeb技术，包括Jsp，Servlet，JDBC

# 二.项目涉及的技术
### 前端：
- 使用了BootStrap框架，展示页面
- JSP处理前端数据
### 后端：
- Servlet处理业务
- JDBC连接数据库
